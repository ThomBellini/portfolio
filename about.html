About Thom


Thom Bellini is a musician, tour manager and 3D artist. He has been playing guitar and bass for over 25 years, and has been composing music for over 15 years.
He has been working as a 3D artist for over 5 years, and has experience with Blender, Fusion 360 and 3d Printing.
He has been touring with a number of live bands since 2010 in a number of different roles.
Now, the next skill He is mastering is coding using HTML, CSS and Python (as you can see by this website you are visiting.)
